# Tugas1_IF4020_Kripto

## Description

Chat web-app with Application Layer Security (ALS), End-to-end Encryption (E2EE), and Digital Signature.

## How To

Duplicate `.env.example`, rename it to `.env`, and change `API_KEY`, then do the usual.

```
npm install
npm run dev
```
